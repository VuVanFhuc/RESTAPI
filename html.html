ĐỂ CHẠY MONGGODB KẾT NỐI FIREBASE VỚI VISUAL CODE TA THỰC HIỆN CÁC BƯỚC SAU :
        1. tạo server.js
        2. tạo model.js
        3. chạy lệnh npm init 
        4. chạy lệnh npm i mongoose
        5. để cài đặt nodemon ta cần chạy lệnh sau : npm install -g nodemon sau đó để chạy ta dùng nodemon server.js
        6. kiểm tra qua cổng http://localhost:3000
